## What is DAU/MAU?

"Dau mau" refers to the DAU/MAU ratio, a key metric in apps/digital products for user engagement, meaning Daily Active Users (DAU) divided by Monthly Active Users (MAU), indicating how "sticky" or frequently users return, with higher percentages showing greater user loyalty and daily use
