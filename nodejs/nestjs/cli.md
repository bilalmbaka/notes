# COMMAND LINE

### Installing NestJS

```bash
# installs it globally

$ npm install -g @nestjs/cli

#Alternatively, you can use this command npx command without installing the cli globally.

$ npx @nestjs/cli@latest
```

### To create a new scaffold project

```bash

$ nest new project-name

```

To quickly create a CRUD controller with built-in validation,
you can use the CLI's CRUD generator:
$ nest g resource [name]
